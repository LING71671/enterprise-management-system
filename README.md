# 企业管理系统前端原型

## 项目概述

这是一个基于HTML、CSS和JavaScript的企业管理系统前端原型，包含以下六个子系统：

1. **生产管理子系统**：生产计划管理、生产任务排产、物料需求计划、库存管理、生产订单管理、质量管理
2. **销售管理子系统**：客户信息管理、销售数据报表、销售订单管理、产品定价和促销管理、销售团队管理
3. **设备管理子系统**：设备状态监控、设备维护计划管理、设备故障记录和分析、设备信息管理
4. **采购管理子系统**：供应商信息管理、采购流程管理、采购订单跟踪、采购数据分析
5. **仓储管理子系统**：仓储基本操作、仓库布局和货位管理、库存预警和盘点、运输计划和跟踪
6. **员工管理子系统**：员工信息管理、招聘和培训管理、考勤和薪资管理、员工绩效评估

## 项目结构

```
├── src/
│   ├── assets/
│   │   ├── js/          # JavaScript文件
│   │   ├── css/         # CSS样式文件
│   │   └── images/      # 图片资源
│   ├── components/      # 共享组件
│   ├── pages/           # 页面文件
│   │   ├── production/  # 生产管理子系统
│   │   ├── sales/       # 销售管理子系统
│   │   ├── equipment/   # 设备管理子系统
│   │   ├── purchase/    # 采购管理子系统
│   │   ├── warehouse/   # 仓储管理子系统
│   │   └── employee/    # 员工管理子系统

├── index.html           # 系统入口文件
├── package.json         # 项目配置文件
└── README.md            # 项目说明文件
```

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **开发工具**：http-server (用于本地开发服务器)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

这将启动一个本地开发服务器，并在浏览器中打开系统。

### 登录系统

- 用户名：任意输入
- 密码：任意输入

## 系统功能

1. **登录/退出**：系统登录和退出功能
2. **子系统切换**：通过侧边导航菜单切换不同的子系统
3. **模块展示**：每个子系统包含多个功能模块，展示系统的基本结构

## 注意事项

- 这是一个前端原型，使用模拟数据展示功能
- 系统不包含后端功能，仅用于展示前端界面和交互流程
- 响应式设计，适配不同设备屏幕
